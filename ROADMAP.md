## 이력서 콘텐츠 예시

### 기본 정보

```
이름: 홍길동
직책: 프론트엔드 개발자
이메일: hong@example.com
GitHub: github.com/honggildong
```

### 자기소개

```
안녕하세요, 사용자 경험을 중시하는 프론트엔드 개발자 홍길동입니다.
웹 기술을 활용하여 직관적이고 아름다운 인터페이스를 만드는 것을 좋아합니다.
```

### 기술 스택

```
Frontend: HTML, CSS, JavaScript, React, Vue.js
Styling: TailwindCSS, SCSS, Styled-Components
Tools: Git, VS Code, Figma
```

### 경력

```
ABC 테크 | 프론트엔드 개발자 | 2022 - 현재
XYZ 스타트업 | 웹 개발자 | 2020 - 2022
```

### 학력

```
OO대학교 | 컴퓨터공학과 | 2016 - 2020
```

### 프로젝트

```
1. 포트폴리오 웹사이트 - 개인 이력서 및 프로젝트 소개
2. 할일 관리 앱 - React 기반 Todo 애플리케이션
3. 날씨 대시보드 - API 연동 날씨 정보 표시
```
